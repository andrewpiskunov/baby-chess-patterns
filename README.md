# Baby Chess Patterns

Высококонтрастные чёрно-белые узоры с шахматной темой для развития зрения
младенца 0–3 месяцев: фокусировка взгляда и слежение за медленно движущимся
объектом.

Это **не** обучение шахматам — в этом возрасте оно невозможно. Это визуальная
стимуляция, оформленная в шахматной эстетике «на вырост».

## Запуск

Откройте `index.html` в браузере. Зависимостей и сборки нет.

## Сцены

| Клетка | Сцена | Что происходит |
|---|---|---|
| a1 | Доска | Крупная шахматная доска, медленная инверсия цветов |
| b2 | Фигура | Силуэт фигуры (пешка → ладья → конь → слон → ферзь → король) медленно плывёт по экрану; каждая фигура называется вслух |
| c3 | Узоры | «Бычий глаз» из колец поверх полос |
| d4 | Спираль | Медленно вращающаяся архимедова спираль |
| e5 | Диагонали | Диагональные полосы, ползущие поперёк себя |
| f6 | Точки | Сетка крупных точек с медленным дрейфом |

## Управление

- Клик по экрану / стрелки ← → — смена сцены
- Пробел — пауза
- На сцене «Фигура» (и только там): выбор фигуры — по кругу или одна
  конкретная (♟♜♞♝♛♚), «Инверсия» — белая фигура на чёрном фоне,
  «Звук» — озвучка названия фигуры с повтором каждые 5 секунд
  (Web Speech API, русский голос системы)
- Скорость: статика / очень медленно / медленно
- ✕ — скрыть панель; вернуть — коснуться экрана или нажать клавишу
- Таймер сессии 3 или 5 минут — по окончании экран плавно гаснет
- «Во весь экран», «Печать карточек», ⓘ — справка
- Настройки (сцена, фигура, скорость, звук, инверсия) сохраняются между открытиями

## На телефоне и офлайн

Приложение — PWA: при открытии по HTTP(S) регистрируется service worker,
после первого визита работает без сети. На телефоне можно добавить на
главный экран («Добавить на экран „Домой“») — откроется на весь экран,
как обычное приложение.

## Печать карточек (рекомендуемый способ)

Американская академия педиатрии не рекомендует экраны детям до 18 месяцев.
Кнопка «Печать карточек» печатает те же паттерны на A4 — показывайте
карточки с расстояния 20–30 см, по 3–5 минут, когда малыш спокоен и бодр.
